---
title: Variable Stars (PAP313)
---

Analysis and modelling of variable star photometry.

- **Period**: 1, 2
- **Content**: 14 lectures, Assignment(-s) each week
- **Themes**: Theory 5%, Methods 70%, Observations 15%, Phenomena 10%
- **Teachers**: [[Lauri Jetsu]]

---
## Content

- Discrete Chi-square Method (DCM)
- Discrete Fourier Transform (DFT)
- DCM and DFT applications to real variable star data
- DCM: theory, programming and application
- DCM and DFT performance comparison
- DCM predictive multiperiodic models: theory and application
- Python coding
- Chromospherically active stars: Differential rotation, active longitudes, activity cycles
- Binary star O-C data: Light time effect, Applegate effect, Apsidal motion, Mass transfer

- [[Tieteellinen laskenta I]]
- equivalent Python programming skills
