---
title: Time Series Analysis in Astronomy (PAP312)
---

Methods and programming for time series analysis of astronomical data.

- **Period**: 1
- **Content**: 14 lectures, Assignment(-s) each week
- **Themes**: Theory 5%, Methods 85%, Observations 5%, Phenomena 5%
- **Teachers**: [[Lauri Jetsu]]

---
## Content

- Python coding
- Discrete Chi-square Method (DCM)
- Discrete Fourier Transform (DFT)
- DCM: Theory, programming, application
- DCM and DFT application to simulated data
- DCM and DFT performance comparison
- Time Series Analysis, Simulation of astrophysical data, Linear and non-linear models
- Multiperiodicity, DFT pre-whitening, De-trending, Bootstrap, Fisher-test, Nested models

- [[Tieteellinen laskenta I]]
- equivalent Python programming skills
